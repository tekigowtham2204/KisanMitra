# KisanMitra (किसानमित्र)
KisanMitra is an open-source multilingual PWA for Indian farmers with live mandi prices, MSP intelligence, weather advisories, government schemes, and direct market access.

## Features
- **Real-time Mandi Prices**: Track commodity prices across mandis in India.
- **Weather Alerts**: Location-specific 7-day weather forecasts and crop advisories.
- **MSP Tracker**: Track Minimum Support Prices for 23+ crops.
- **Government Schemes**: Information and access to 50+ government schemes.
- **Direct Market Access**: Connect with buyers directly.

## Usage
Simply open `index.html` in your browser, or run a local server:
```bash
npx http-server
```

## Technologies
- HTML5
- CSS3 (Custom Variables)
- Vanilla JavaScript
- PWA Support (Service Worker)
